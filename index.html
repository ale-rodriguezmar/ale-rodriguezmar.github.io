Implementacion de Realidad Aumentada

¿Qué es el proyecto?

Este proyecto explora la implementación de realidad aumentada utilizando AR.js, permitiendo la integración de modelos 3D en entornos web sin necesidad de aplicaciones externas.

Herramientas Utilizadas

  • JavaScript
  • Blender
  • HTML y CSS
  • AR.js

Problemas y Soluciones

Generación del modelo en Blender

Se enfrentaron desafíos en la exportación de modelos 3D optimizados para la web, solucionados mediante la utilización del formato glTF.

Autenticación HTTPS

La implementación de HTTPS fue esencial para el funcionamiento de AR.js, lograda mediante la configuración adecuada del servidor.

Ubicación del modelo

Se ajustaron las coordenadas y escalas de los modelos para una correcta visualización en entornos de realidad aumentada.
